# ConnectHub

ConnectHub is the application being developed in this repository: a production-quality educational Cloud Native project that demonstrates modern backend engineering and Platform Engineering practices.

The repository is intended to showcase how a real-world system can be designed, developed, containerized, deployed, and observed across local and platform environments.

## Technology snapshot

- Java 21
- Spring Boot 3
- Spring WebFlux
- Go
- Python
- Bash
- MongoDB
- Docker
- Kubernetes
- OpenShift
- Keycloak
- Prometheus
- Grafana

## Project goals

- Build a portfolio-grade cloud native system with clear architecture and documentation.
- Demonstrate service decomposition, secure authentication, observability, and deployment automation.
- Keep the repository suitable for public review, learning, and iterative expansion.
- Treat infrastructure and developer experience as first-class concerns.
